## Authentication

All API endpoints are authenticated using Bearer tokens. You can obtain your API key from the [SLCT.AI Dashboard](https://dashboard.slct.ai).

```bash
curl -X GET "https://api.slct.ai/api/documents" \
  -H "Authorization: Bearer your-api-key"
```

<Note>
  Keep your API key secure and never expose it in client-side code.
</Note>

## Quick Start

Here's a simple example to get you started:

```typescript
import SlctClient from '@slct/api-client';

// Initialize the client
const client = new SlctClient('your-api-key');

// Upload a document
const documentId = await client.uploadFile(file);

// Wait for processing
const result = await client.waitForProcessing(documentId);
```

## API Structure

Our API is organized around three main concepts:

<CodeGroup>
  ```bash Documents
  GET /api/documents         # List all documents
  POST /api/documents/new    # Create new document
  GET /api/documents/:id     # Get single document
  ```

  ```bash Processing
  Status: processing -> completed
  Results include:
  - Extracted text
  - Structured data
  - Confidence scores
  ```

  ```bash Webhooks
  Events:
  - document.created
  - document.processing
  - document.completed
  - document.failed
  ```
</CodeGroup>
